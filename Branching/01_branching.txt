-.-What is Branching:

Branching in 'C' refers to the decision-making process where the flow of 
program execution changes based on certain conditions. It allows a 
program to take different paths instead of executing instructions 
sequentially. The most common branching statements in 'C' are 
if, if-else, if-else if ladder, and switch. Using these, a program can 
check conditions and execute specific blocks of code accordingly. 
For example, with if-else, the program decides between two alternatives, 
while the switch statement is useful when there are multiple options 
to choose from. Branching makes programs flexible, logical, and 
capable of solving real-life decision-based problems.

-.- Structure of Branching: 

1. if statement :

   if (condition) {
       // statements
   }

2. if-else statement :

   if (condition) {
       // statements if condition is true
   }
   else {
       // statements if condition is false
   }

3. if-else if ladder :

   if (condition1) {
       // statements if condition1 is true
   }
   else if (condition2) {
       // statements if condition2 is true
   }
   else {
       // statements if none of the above are true
   }

4. nested if :

   if (condition1) {
       if (condition2) {
           // statements if both are true
       }
   }

5. switch statement :

   switch (expression) {
       case value1:
           // statements
           break;
       case value2:
           // statements
           break;
       ...
       default:
           // statements if no case matches
   }
